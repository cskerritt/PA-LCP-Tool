# Examples

`sample_life_care_plan.xlsx` is a complete, generated workbook built from the
**synthetic** sample inputs bundled at `src/palcp/data/` (a fictional 42-year-old
claimant, 38-year horizon, 14 care items across nine categories). It is provided
so you can open the finished, formatted output without running anything.

Regenerate it any time with:

```bash
palcp sample --out examples/sample_life_care_plan.xlsx
```

All figures, names, and rates in this example are illustrative only and must not
be relied upon. The growth/discount rates carry an `ILLUSTRATIVE` source tag.
